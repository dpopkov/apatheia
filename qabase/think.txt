План
====
Создать модуль qabase (Questions-Answers Base)
Реализовать п.1 и 1.1


Варианты Использования
======================

1. Занесение списка Вопросов
----------------------------
У меня есть список вопросов. Я просто заношу все вопросы из списка
CREATE TABLE questions (
    id serial,
    wording text,
    category_id
);
Вопросу не нужен источник, источник нужен ответу

Нужен список категорий:
Base
OOP
Collections
Java8
Algo
InputOutput
SQL
JDBC
GC
Servlets
JSP
Multithreading
Hibernate
Spring

1.1 При ответе даю свой вариант ответа (Попытка Ответа, может быть много)

2. Занесение Вопрос-ПравильныйОтвет   (у Ответа: статус, источник)

3. Поиск по Вопросам

4. Добавление Вариантов Ответов с указанием источников, ссылок на источники

5. Связывание близких Вопросов (по ключевому термину)



Последовательность Работы
-------------------------

1. Занести Вопросы

2. Пытаться дать на них ответы

3. Найти правильные ответы

4. Повторить
